喜欢markdown写作的人有福了，整理种类超多的markdown编辑器列表，包含了本地剪辑器和在线编辑器和公众号排版工具，关键还都是免费可用的。还有没列出来的希望大家补充。

软件类

安装版或者免安装版的markdown编辑器客户端，全部都是免费。

MarKing

  • 开源支持中文
  • 支持所见即所得和编辑预览双窗口模式切换
  • 本地多库管理
  • 可视化表格编辑器
  • 双链系统与知识图谱
  • Excalidraw 手绘白板
  • 自动防抖保存与历史快照
  • 支持导出word、pdf、html
  • windows\macOS\Linux平台支持



    官网：https://markingmd.com/

Markar

  • 开源、支持中文
  • 支持文件列表和大纲
  • 直接存储本地图片同目录下的assets文件夹
  • 支持ai，需要在设置页面自定义api key


    官网：https://markra.app/zh-CN/

MarkText

  • 开源，支持中文
  • 支持文件列表和大纲
  • 所见即所得，体验类似Typora
  • 图片支持本地和云端（Picgo、Github、自定义命令行脚本）
  • windows\macOS\Linux平台支持


    官网：https://mark-text.cn/index.html

SoloMD

  • 开源，支持中文
  • 轻量，< 15MB
  • 支持macOS、Windows、Linux、Android、iOS
  • 支持同步到GitHub仓库
  • 支持导出pdf
  • 支持实时预览和分页模式
  • 集成AI，需填写自己的密钥



    官网:https://solomd.app/zh/

ghostwriter

  • 开源，支持中文
  • 支持macOS、Windows、Linux

    官网：https://ghostwriter.kde.org/zh-tw/

markdownpad

老牌markdown编辑器，看下界面吧，不支持中文

官网：http://markdownpad.com/

Texts

实时预览，英文。

官网：http://markdownpad.com/

在线类

用完即走的免费的在线编辑器。

作业部落

很久之前就用过，体验最好，支持无登录使用，也支持注册云端保存。
编辑器自带的语法是真的很多很详细，可以用来做markdown语法词典了。
支持导出markdown、html、pdf(需登录)。

网址：https://www.zybuluo.com/

markdown.com.cn

  • 支持文件列表
  • 浏览器缓存，下次打开还有上次编写的文件
  • 支出导入导出markdown文件、导出html文件


网址：https://markdown.com.cn/

公众号排版型

主要是用来给公众号排版的markdown在线编辑器。都是可以免费使用的。

Doocs MD

专为中文排版优化，多种主题模板，一键复制富文本到微信公众号/知乎/掘金，支持数学公式和 Mermaid，写推文非常方便。

  • 支持导出导入md文件，导出HTML、PDF、png图片。
  • 支持自定义css样式
  • 支持定义模板
  • 支持浏览器缓存，清空缓存后配置会重置

    网址：https://md.doocs.org/

markdown.com.cn

markdown.com出品的公众号排版工具。

  • 主题丰富
  • 支持导出导入md文件，导出word。
  • 支持浏览器缓存，清空缓存后配置会重置
  • 支持缓存的历史版本恢复


    网址：https://markdown.com.cn/wechat/

Arya

markdown.com出品的公众号排版工具。

  • 开源
  • 支持导出导入md文件，导出Word、PDF、png图片。
  • 支持文件列表
  • 支持浏览器缓存，清空缓存后配置会重置

    官网：https://markdown.lovejade.cn/

花生公众号排版器

  • 开源
  • 支持从任何地方复制粘贴图片（截图、浏览器、文件管理器）
  • 支持多图排版
  • 使用 IndexedDB 持久化，刷新不丢失
  • 支持历史版本恢复

    网址：https://editor.huasheng.ai

Raphael Publish

  • 开源
  • 支持从任何地方复制粘贴图片（截图、浏览器、文件管理器）
  • 多图排版
  • 支持导出为 PDF 和 HTML 文件

    网址：https://publish.raphael.app/
    github：https://github.com/liuxiaopai-ai/raphael-publish

Paper

  • 开源
  • 支持信息卡片

    网址：https://paper.thus.chat/
    github：https://github.com/xiaolinbaba/Paper

知识库

知识库一般是支持markdown语法的，但是不是直接编辑md文件，大多数知识库都是自己的文件格式。

obsidian

第一个必须是大名鼎鼎的obsidian了。
个人使用免费，本地纯文本存储，支持双向链接、知识图谱及丰富的社区插件，Markdown 渲染效果好，适合构建第二大脑。

官网：https://obsidian.md/zh/

Joplin

开源跨平台笔记软件，支持端到端加密、Markdown 编辑及与 Dropbox/OneDrive/WebDAV 同步，适合需要多端同步的用户。

官网：https://joplinapp.org/cn/

开发者

开发者的话ide的markdown插件就可以直接用。

Visual Studio Code

装 _Markdown All in One_或 _Markdown Preview Enhanced_插件即可获得目录、实时预览、导出 PDF 等功能，适合已在用 VS Code 的开发者。

文本编辑器

notepad++、edit等文本编辑器也是直接编辑器md文档的，只是没有实时预览功能而已。
