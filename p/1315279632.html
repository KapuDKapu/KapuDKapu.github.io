【网站分享】windows系统下载站--吻妻

今天发现个网站，完全是被网站UI所吸引，然后才关注的内容。
一个系统分享网站，原先是专注于win7的系统优化的，谐音梗“win7”=“吻妻”。win7的站点还在运行，这里主要介绍的还是win10的，毕竟现在基本都习惯用win10了。

主要是提供两个版本：专业版window10 pro和企业版windows10 ltsc。专业版只有64位，企业版有32位和64位下载。镜像类型的话是WIN和GHO。下载方式支持迅雷、百度网盘、微云盘、360云盘、迅雷云盘、天翼云盘。



版本特色

  • 基于原版Win10 企业版LTSC优化制作
  • 原生Internet Explorer 11
  • 原生无捆绑微软UWP全家桶
  • 原生无应用商店、小娜、OneDrive、Edge浏览器
  • 移除SmartScreen 筛选器、Windows 安全中心
  • 移除Xbox及其相关组件
  • 移除零售演示内容和绝大部分可选功能
  • 增加仿宋GB2312、楷体GB2312、方正小标宋3种字体
  • 支持在线更新（自行开启更新服务）
  • 支持固态硬盘4K对齐、支持NVMe M.2固态
  • 支持锐龙、八代、九代、十代等新平台完美安装
  • 支持全自动无人值守安装与自建帐户安装（WIM镜像）
  • 集成全智能万能驱动助理（仅GHO镜像集成）
  • 提供WIM和GHO两种镜像下载

运行库支持

  • DirectX 9.0C+DirectX 11
  • Microsoft .NET Framework 3.5
  • Microsoft .NET Framework 4.7（自带）
  • VB VC++ 2005/2008/2010/2012/2013/2019

安装方法

1、硬盘下安装方法：（适用于MBR分区）

调用系统安装器选中GHO系统镜像即可安装系统。将本GHO镜像文件与”硬盘安装器.exe”工具 一起复制到D盘根目录（非安装盘均可），运行”硬盘安装器.exe “，点击”确定”进入系统自动重 启安装过程，直到完成。

2、 PE下安装方法：

用光盘或U盘引导进PE系统，运行安装器调用GHO/WIM系统镜像即可安装系统。
甚至还贴心的提供了定制好的启动盘根据连接购买即可。
