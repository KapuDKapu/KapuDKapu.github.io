1、简介

Jupyter Notebook 的本质是一个 Web 应用程序，便于创建和共享文学化程序文档。支持运行 40 多种编程语言，支持实时代码，代码高亮，，数学方程，可视化和 markdown。
你可以把它理解成一个网页版的代码编辑器。

2、安装

安装Jupyter Notebook前需要先安装python，因为它依赖用python。
官网给出两种安装方式：

  • 1、Anaconda
    安装最新的Anaconda即可，因为Anaconda集成了Python，Jupyter Notebook以及其他常用的科学计算和数据科学包。
  • 2、pip
    需要先安装python和pip，然后使用pip命令进行安装，安装命令如下

Text File

Folder

文件夹的话就是在Jupyter工作空间的文件夹下新建一个文件夹，步骤是先使用默认名字建好文件夹然后再重命名。

Terminal

新建Terminal是在网页打开一个命令窗口，你是在windows下运行的Jupyter Notebook的话，打开的就是cmd窗口，在linux上运行Jupyter Notebook的话打开的就是shell窗口。
